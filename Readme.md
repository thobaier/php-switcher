# Linux PHP-Switcher
Locates and lists the currently installed PHP-Versions of a Linux and let's you switch between them.

Usage:
```bash
sh ./phpswitcher
# or
bash ./phpswitcher
```

Requirements:
- sudo privileges
- update-alternatives
- sed

Compatibility:
- Shell
- Bash

Tested:
- Ubuntu Terminal
- ZSH
- Terminator
- Xterm
- Tilix
